## ğŸ“Š **Crossover Hiring Process Overview**

### **5-Step Assessment Process:**

1. **Basic Fit Screening** (5 minutes)
   - Background and experience questions

2. **CCAT Test** (15 minutes)
   - 50 questions covering math, verbal, and spatial reasoning
   - Need 85-90th percentile (35-38+ correct answers)
   - **Major elimination point**

3. **English Communication** (2 minutes)
   - Audio recording demonstrating English fluency
   - **Another major elimination factor**

4. **Technical Skills Assessment** (40 minutes - 3 hours)
   - HackerRank-style coding challenges
   - Automatically graded
   - Tests algorithms and data structures

5. **Project Evaluation** (72 hours)
   - Real-world coding project
   - Evaluated on design, code quality, and completeness
   - Can request extension if needed

### **Interview Rounds (If Selected):**
- VP of Engineering (30 mins)
- SVP of Engineering and Global Ops (30 mins)
- CEO (30 mins)

**Success Rate:** Only ~5-10% of applicants get hired
